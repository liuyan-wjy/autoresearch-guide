# autoresearch-guide

Static site for an unofficial AutoResearch guide, designed for deployment on Cloudflare Pages.

## Pages

- `/` - main landing page
- `/autoresearch-github-guide/` - GitHub explainer
- `/run-autoresearch-on-mac-or-small-gpus/` - hardware guide
- `/autoresearch-vs-autora/` - disambiguation page

## Deploy

This repo is intended to deploy as a static site on Cloudflare Pages with no build step.
