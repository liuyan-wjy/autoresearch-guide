Privacy Policy

Privacy Policy

Open Repo Guide currently publishes static content pages and does not offer user accounts, comments, or a newsletter. This page explains the limited data handling that may still occur when you visit the site.

Data the Site Does Not Intentionally Collect

The site does not currently ask visitors to create accounts, submit forms, or provide payment details. It also does not currently run a public comment system on the site.

Infrastructure Logs

Like most websites, infrastructure providers such as hosting, CDN, and security platforms may process basic request information to deliver the site, prevent abuse, and maintain availability. This can include IP address, user agent, and request timing data.

External Links

Pages on this site link to GitHub and other third-party resources. When you leave Open Repo Guide, those external websites apply their own privacy practices and policies.

Policy Updates

Because Open Repo Guide is currently a simple static publishing site, this policy is intentionally narrow. If the site later adds analytics, affiliate links, email collection, or other user-facing features, this page should be updated to reflect those changes before or at the time they are introduced.

This privacy policy describes the site in its current static publishing state.
